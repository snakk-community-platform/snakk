PRINCIPLES

- Collect minimal data
- Be explicit about consent
- Allow users to leave cleanly
- Prefer anonymization over retention

---

DATA WE COLLECT

- Email address (if authenticated)
- OAuth provider ID (if used)
- Display name / username
- Posts and discussions
- Read/unread metadata
- IP address (temporary, anonymized)
- Cookie-based anonymous user ID

---

DATA WE DO NOT COLLECT

- Real names
- Location data
- Device fingerprinting
- Third-party analytics by default
- Cross-site tracking

---

EMBEDS & THIRD-PARTY CONTENT

- Media embeds are disabled by default
- User must explicitly consent per provider
- Consent can be:
  - One-time
  - Remembered for future visits (if legally allowed)

Before consent:
- Placeholder shown
- No third-party requests made

---

COOKIES

- Essential cookies only by default
- Anonymous user ID cookie
- Auth cookie (if logged in)
- No marketing cookies

Cookie consent banner shown where required.

---

USER RIGHTS

Users can:
- Export their data (JSON)
- Delete their account
- Request anonymization
- Withdraw embed consent

---

ACCOUNT DELETION

On account deletion:
- User account removed
- Posts anonymized
- Display name replaced
- Avatar removed
- Content retained unless explicitly requested otherwise

Rationale:
- Preserve discussion integrity
- Respect right to be forgotten

---

POST DELETION

- Hard delete allowed in short self-delete window
- After window:
  - Soft delete
  - Content hidden
  - Metadata retained for integrity

---

IP HANDLING

- IPs stored only if required for security
- IPs anonymized where possible
- IP retention minimized
- No IPs exposed publicly

---

DATA RETENTION

- No unnecessary long-term storage
- Logs expire automatically
- Backups respect deletion policies

---

TRANSPARENCY

- Clear privacy policy
- No dark patterns
- No silent data sharing

---

NON-GOALS

- Selling user data
- Ad tracking
- Behavioral profiling
